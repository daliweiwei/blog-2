# 条纹材质：StripeMaterialProperty (opens new window)

名称 类型 默认值 是否可选 描述
orientation Property (opens new window) | StripeOrientation (opens new window) StripeOrientation.HORIZONTAL 可选 指定StripeOrientation (opens new window)
evenColor Property (opens new window) | Color (opens new window) Color.WHITE 可选 第一个 color
oddColor Property (opens new window) | Color (opens new window) Color.BLACK 可选 第二个 color
offset Property (opens new window) | Number 0 可选 用于指定起始材料到图案的距离。
repeat Property (opens new window) | Number 1 可选 指定条纹重复的次数。
